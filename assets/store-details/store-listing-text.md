## Post Install Tip (required, max 1500 chars)
To get started with Action Sync:

1. Open any Google Doc
2. Place your cursor where you want to track an action
3. Type @ and select "Create action" from the Action Sync section
4. Fill in the action text, optional assignee (email), and status — then click Create
5. The action chip is inserted at your cursor and synced to your ActionSheet

To view all actions for this document, open the Action Sync sidebar from the Extensions menu.

Hover over any inserted action chip to see its current status and assignee in a preview card.
